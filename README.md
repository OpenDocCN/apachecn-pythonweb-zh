## 下载

### Docker

```
docker pull apachecn0/flygon-pythonweb-zh
docker run -tid -p <port>:80 apachecn0/flygon-pythonweb-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g flygon-pythonweb-zh
flygon-pythonweb-zh <port>
# 访问 http://localhost:{port} 查看文档
```
